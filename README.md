# Sales-Analytics-Database

This project is a simple relational database designed for practicing SQL and data analysis using PostgreSQL.

## Features
- Relational database design (Users, Products, Orders)
- SQL queries for business analytics
- JOIN operations between multiple tables
- Aggregate functions (SUM, COUNT, AVG)
- Grouping and reporting insights

## Technologies
- PostgreSQL
- SQL

## Sample Analysis
- Total revenue calculation
- Best-selling products
- Customer spending analysis
- City-based user distribution

## How to run
1. Create database in PostgreSQL
2. Run schema.sql
3. Run seed_data.sql
4. Execute queries.sql
